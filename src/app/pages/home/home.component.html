Angular BootKit

A component library isn't the only piece of the puzzle for building Angular applications. There are many challenges to overcome, such as form validation, security, localization, file upload/download, and more.
Angular BootKit is an "infrastructure" library based on Bootstrap CSS that helps you focus on your application's core functionality, not these essential prerequisites.

UI Infrustructure

Building Angular apps involves managing a wide range of tasks, from validation and localization to file uploads, authentication, and authorization. Angular BootKit is an infrastructure library that simplifies these common challenges, allowing you to focus on your core application logic.

Pure Angular

No dependency on third-party JavaScript libraries like Bootstrap.js, Popper.js or jQuery. BootKit is a pure Angular library built on top of @angular/cdk and Bootstrap CSS. We've removed Bootstrap's JavaScript dependencies.

Components

Build your application with our lightweight, feature-rich components. Designed to meet enterprise-level needs, they handle a wide range of UI complexities. Focus on your core business logic.

CSS Neutral

BootKit components are built on top of Bootstrap CSS. Customize the look and feel of your components by simply customizing Bootstrap's CSS. We've used only Bootstrap styles in our components.

Light

Our components are designed with simplicity and performance in mind. By using our library's components, you can selectively include only the necessary Bootstrap styles, minimizing bundle size and improving loading times.
