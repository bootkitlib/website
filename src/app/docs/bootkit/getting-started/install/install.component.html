The only required dependency is Angular CDK.
Install @bootkit/angular and Angular CDK libraries via npm:

ng add @angular/cdk
npm install @bootkit/angular
Important:
Bootkit does not depend on Bootstrap.js, jQuery, Popper.js, or any other JavaScript library. Its only dependency is Angular CDK.

The Angular CDK (Component Development Kit) is a library developed by Google that provides a set of tools and utilities specifically designed for building and enhancing Angular component libraries.
